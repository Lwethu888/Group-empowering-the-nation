Empowering the Nation is a dual platform initiative that includes both a responsive website and a mobile application. It was created to uplift South Africans by providing accessible, affordable, and practical education. The platform features life-enhancing courses such as Life Skills and Landscaping, with a focus on reaching youth, women, and marginalized communities. Its core mission is to promote independence, improve personal well-being, and support national development through inclusive digital learning. This project was developed as part of the Work Integrated Learning program and demonstrates a strong commitment to social impact, technical excellence, and polished presentation.

Key Features and Functionality
Website
- Landing Page: Introduction to the initiative and navigation to core sections
- Courses Page: Scrollable catalog of life-enhancing courses with images and descriptions
- Course Details Page: Individual course breakdown including pricing and benefits
- Contact Page: Embedded map, contact details, and feedback form
- Footer: Social media icons and copyright notice
  
Mobile App
- Home Screen: Brand message and course navigation
- Course Details Screen: Full course descriptions with images and pricing
- Login Screen: Simulated login for demonstration purposes
- Navigation: Stack-based routing with reusable components
- Footer Icons: Social media and contact options for mobile users
